# FUTURE_DS_03
## Marketing Funnel & Conversion Performance Analysis

### About
Analysis of Olist marketing funnel data to identify 
conversion drop-offs, channel performance and opportunities 
to improve lead-to-customer conversion.

### Tools Used
- Python (pandas, numpy, matplotlib)
- Jupyter Notebook

### Dataset
Olist Marketing Qualified Leads Dataset from Kaggle

### Key Findings
- Overall conversion rate is 10.52% with 89.48% drop-off
- Paid search converts best at 12.30%
- Home decor is top business segment (103 deals)
- Conversions peaked in April 2018 then declined
- Online medium businesses convert most (328 deals)
